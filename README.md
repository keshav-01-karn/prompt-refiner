# Prompt Refiner – AI Prompt Optimization Tool

An interactive web tool that transforms rough, vague, or simple prompts into optimized prompts designed to generate better responses from large language models.

The application helps users improve prompt quality by applying prompt engineering techniques automatically using AI.

Built with Python, Gradio, and Gemini API.

## Features

- Refine vague prompts into structured prompts
- Improve prompt clarity and specificity
- Interactive web interface
- Example prompts for quick testing
- Works with Google Gemini models
- Lightweight and easy to run locally

## Example

**Original Prompt**
Write about climate change

**Refined Prompt**
Explain the causes, impacts, and potential solutions for climate change in a clear and structured manner suitable for a general audience.


## Tech Stack

- Python
- Gradio
- Gemini API
- python-dotenv

## Use Cases

-Learning prompt engineering
-Improving prompts before sending them to AI models
-Experimenting with prompt optimization
-Understanding how prompt structure affects LLM outputs
